تحديث جديد لـ Windows 11

هذا هو الخيار الأفضل لتثبيت Windows 11 على الجهاز الذي تستخدمه حاليًا. انقر فوق تنزيل الآن للبدء. ملاحظة: لا يعمل مساعد التثبيت بنظام Windows 11 على أجهزة الكمبيوتر الشخصية بمعالجات Arm؛ ولكن يعمل فقط لمعالجات x64.إذا كنت تريد إجراء عملية إعادة تثبيت أو تثبيت نظيف لنظام Windows 11 على كمبيوتر شخصي جديد أو مستعمل، فاستخدم هذا الخيار لتنزيل أداة إنشاء الوسائط لجعل USB أو DVD قابلا للتشغيل. ملحوظة: لا يمكن استخدام أداة إنشاء الوسائط بنظام Windows 11 لإنشاء وسائط التثبيت الخاصة بأجهزة الكمبيوتر الشخصية بمعالجات Arm؛ ويمكنها إنشاء وسائط لمعالجات x64 فقط.
